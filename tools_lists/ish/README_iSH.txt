iSH APP LIMITATIONS:
-------------------
iSH (iOS Shell) has limited package availability because:
1. It's Alpine Linux based (apk package manager)
2. iOS security restrictions
3. Limited storage/memory

Available package search: apk search [package]
Install manually: apk add [package]

Some tools can be installed via:
- pip3 (Python packages)
- git clone (from GitHub)
- Manual compilation

For more tools, consider using:
- Kali Linux (full features)
- Termux (Android, more packages)
